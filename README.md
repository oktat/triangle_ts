# Triangle

Calc triangle area in TypeScript.

## Install dependencies

```bash
cd triangle_ts
pnpm install
```

If you don't have [pnpm](https://pnpm.io/) installed, you can install it with:

```bash
npm install -g pnpm
```

## Build

TypeScript build.

```bash
npm run pub
```

## Start server

Start server:

```bash
npm start
```
